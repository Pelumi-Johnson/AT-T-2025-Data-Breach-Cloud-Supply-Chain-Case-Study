## Overview
This repository contains a presentation analyzing the 2025 AT&T data breach
that affected over 86 million customers. The incident highlights risks
introduced by third-party cloud providers and weak identity controls.

## Key Topics Covered
- Cloud supply chain compromise (Snowflake)
- Credential-based access without MFA
- Exposure of PII and SSNs
- Organizational and public response
- Security lessons for enterprises

## Why This Matters
This breach demonstrates how attackers often bypass large organizations
by exploiting weaker vendor security, reinforcing the importance of
vendor risk management and strong authentication controls.
